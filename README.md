# 🚀 LunarLander Reinforcement Learning using DQN

## 📌 Project Description
This project implements a Reinforcement Learning agent to solve the LunarLander-v3 environment using the Deep Q-Network (DQN) algorithm from Stable-Baselines3.

The agent learns how to land a spacecraft safely on the landing pad by interacting with the environment and maximizing rewards.

---

## 🎯 Objectives
- Learn Reinforcement Learning concepts.
- Train an agent using the DQN algorithm.
- Test the trained model.
- Visualize the LunarLander environment.

---
